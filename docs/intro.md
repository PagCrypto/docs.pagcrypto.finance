# Bem-vindo à **PagFinance**

:::tip[GPag]

Aproveite sem moderação

:::

A **PagFinance** é a infraestrutura de pagamentos Web3 que conecta stablecoins e criptomoedas a pagamentos em moeda local, como PIX, boletos e transferências bancárias.

## 🚀 O que é a **PagFinance**?

A **PagFinance** permite que pessoas e empresas realizem pagamentos do mundo real utilizando suas carteiras descentralizadas, com suporte a stablecoins como USDC e XRP.

Com nossa plataforma, é possível:
- Pagar boletos e PIX diretamente com CRIPTO.
- Integrar sistemas via API ou SDK.
- Automatizar liquidações usando contratos inteligentes.
